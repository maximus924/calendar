鍵盤快捷鍵

  • T：回到今天
  • G：跳到日期輸入欄
  • Alt + ←：前一天
  • Alt + →：後一天
  • Alt + PageUp：上個月
  • Alt + PageDown：下個月
  • Ctrl + S：儲存
  • Ctrl + Shift + A：新增空白時段
  • Ctrl + Shift + T：下載 TXT
  • Ctrl + Shift + P：下載 PDF
  • ?：開啟快捷鍵說明

快捷鍵與操作說明

可使用 Tab 在控制項之間移動；在月曆日期上可用方向鍵移動日期，按 Enter 選取日期。輸入內容會自動儲存在本機瀏覽器。

  • T：回到今天。
  • G：跳到日期輸入欄。
  • Alt + 左右方向鍵：前一天、後一天。
  • Ctrl + S：立即儲存。
  • Ctrl + Shift + A：新增空白時段。
