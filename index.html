0SEL 情緒覺察與所在地天氣

每次開啟行事曆時，先花 30 秒覺察自己的情緒，再安排待辦事項。快捷鍵：Alt + 0。

選好情緒後，右側會出現對應的情緒圖像；可搭配 NVDA 線性往下操作。

2新增待辦事項

請依序操作：日期 → 可自行輸入的時段 → 待辦事項內容。待辦事項內容欄按 Enter 只會換行；完成後請按「新增到月曆」。

資料會自動儲存在這台電腦的瀏覽器中。

3當月月曆

快捷鍵：按 G 跳到日期欄。月曆內按 ↑ / ↓ 往上週 / 下週，並報讀整週摘要。

4輸出選項

可下載 TXT、PDF、JSON 備份；也可載入 JSON 備份。ICS 可到 Google 日曆的「設定 → 匯入與匯出 → 匯入」載入。

鍵盤快捷鍵

  • Alt + 0：SEL 情緒覺察與所在地天氣
  • Alt + 1：第一層，今天日期與目前規劃
  • Alt + 2：第二層，新增待辦事項
  • Alt + 3：第三層，當月月曆
  • Alt + 4：第四層，輸出選項
  • Alt + ← / →：前一天 / 後一天
  • Alt + ,：上個月
  • Alt + .：下個月
  • ↑ / ↓：月曆內往上週 / 下週，並報讀整週摘要
  • Ctrl + S：儲存
  • Ctrl + Shift + T：下載 TXT
  • Ctrl + Shift + P：下載 PDF
  • Ctrl + Shift + I：下載 ICS 載回 Google 日曆
  • ?：開啟快捷鍵說明
  • T：回到今天
  • G：跳到日期輸入欄
  • F2：跳到待辦事項內容欄

快捷鍵與操作說明

本頁保留快捷鍵清單中列出的操作；新增待辦請使用「新增到月曆」實體按鈕。每次開啟可先按 Alt + 0 進行 SEL 情緒覺察。
